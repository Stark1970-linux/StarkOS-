# StarkOS-
StarkOS — дистрибутив на основе Arch Linux с собственным консольным установщиком starkos-install, предустановленным набором приложений для работы, графики, кода и безопасности. Live‑система запускается с LightDM для удобства, установленная система — без него (чище и легче). Подходит для тех, кто хочет получить готовую среду без долгой настройки, но не боится консоли.

Особенности

1. Свой установщик, а не чужой

· Не archinstall, не Calamares, а starkos-install — написан руками под конкретные задачи.
· Понятные шаги (1/7, 2/7…), цветной вывод, защита от случайного стирания диска.
· Всё в одном скрипте: разметка, копирование, настройка, GRUB, initramfs — без магии.

2. Два режима графики — чисто и гибко

· Live‑система → LightDM (автологин, удобно для установки и тестов).
· Установленная система → без LightDM, XFCE стартует через .bash_profile.
· Никаких лишних сервисов в целевой системе.

3. Кастомизация без потери возможности сменить

· Обои и фон GRUB лежат в /usr/share — системные, но пользователь может их переопределить.
· Брендинг везде: os-release, hostname, GRUB, LightDM, приветствие.

4. Предустановленный софт — готово к работе сразу

· Офис: LibreOffice Fresh
· Графика: GIMP, Blender
· Код: VS Code, Git
· Интернет: Firefox, Thunderbird
· Мультимедиа: VLC
· Безопасность: nmap, wireshark, hydra, john

5. Установка на реальное железо работает

· Протестировано на i3-9100F, GT 710, 8 ГБ ОЗУ, HDD.
· GRUB встаёт на UEFI, XFCE запускается, мышь работает.

6. Полная прозрачность сборки

· Никаких «бинарных блобов», всё собирается из исходников через archiso.
· Ты можешь взять ISO, разобрать, понять, как он сделан, и повторить.

7. Честная документация без маркетинга

· Написано так, как есть: «StarkOS — это Arch Linux, собранный под себя, с удобным установщиком и предустановленным софтом».
· Никаких «уникальных инновационных технологий», только реальные преимущества.

Что дальше? (дорожная карта)

StarkOS не останавливается на достигнутом. В следующих версиях появятся функции, которые сделают её по-настоящему уникальной и ещё удобнее:

StarkOS v2 (в разработке)

· Proton / Steam — запуск Windows-игр из коробки
· Менеджер точек восстановления — откат системы после обновлений или экспериментов
· Умный установщик драйверов — автоопределение видеокарты и выбор: игровые / профессиональные драйверы
· VPN-менеджер — WireGuard / OpenVPN / ProtonVPN на выбор при установке
· Оптимизация системы — sysctl, zram, планировщик I/O, irqbalance

StarkOS v3

· Свой репозиторий пакетов — темы, утилиты, обновления через pacman
· Свой ISO-билдер — полный контроль над сборкой дистрибутива
· Web-дашборд для установки — управление установкой через браузер (как у SteamOS)

StarkOS v4

· Поддержка ARM — Raspberry Pi, Orange Pi и другие платы


Что важно подчеркнуть

StarkOS v1 — это прочный фундамент.
Уже сейчас она удобна, стабильна и готова к работе.
Но настоящая уникальность придёт с v2, v3 и v4, когда мы добавим функции, которых нет в других Arch-дистрибутивах.
